Scripts used to run the PRScs:

### run_prs.bash

Runs PRS-CS (`PRScs.py`) for multiple phenotypes to generate posterior SNP effect sizes for polygenic risk score calculation.  
The script defines the reference LD panel (`ldblk_1kg_eur`), target sample `.bim` file, GWAS summary statistics files, and GWAS sample sizes for each phenotype, then executes PRS-CS iteratively for each trait, producing posterior effect size files for downstream PRS analysis.

To concatenate the PRS-CS result files, the following Bash command is used for each trait:

`cat Pheno_pst_eff_a1_b0.5_phiauto_chr* > Pheno_pst_eff_a1_b0.5_phiauto_chr_all.txt`

This command merges all chromosome-specific posterior effect size files generated by PRS-CS into a single file (`*_chr_all.txt`), which is then used as input for PRS scoring with PLINK.

### run_plink.bash

Runs PRS scoring using PLINK for multiple phenotypes based on the posterior SNP effect sizes generated by PRS-CS.  
The script uses the target genotype dataset (`--bfile`) and the PRS-CS output files (`*_pst_eff_a1_b0.5_phiauto_chr_all.txt`) to calculate individual polygenic risk scores with the `--score` parameter, generating one PRS profile output for each phenotype.
