# Supplementary Material Repository

This repository contains the supplementary scripts used in the analyses conducted for the article.

The main purpose of this repository is to provide reproducible and organized access to the computational workflow used in the study, including summary statistics standardization, quality control (QC), linkage disequilibrium score regression (LDSC), and polygenic risk score (PRS) analyses.

## Repository Structure

### Standardization&QC

Contains scripts for preprocessing GWAS summary statistics, including standardization of column formats, allele harmonization, rsID conversion, frequency calculations, and general quality control filtering before downstream analyses.

### LDSC

Contains scripts used to run Linkage Disequilibrium Score Regression (LDSC), including summary statistics munging, genetic correlation analyses, result extraction, and correlation matrix visualization.

### PRScs/Run_Scripts

Contains scripts used for Polygenic Risk Score analyses with PRS-CS and PLINK, including posterior SNP effect size estimation, file concatenation, and PRS score calculation in the target sample.

This repository is intended as supplementary material to support the methods and reproducibility of the analyses described in the manuscript.
