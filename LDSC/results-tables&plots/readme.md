This section content the scripts files responsable to transform the LDSC logs results in tables and correlation matrix. Follow the order:

### new_table.bash

Reads all `.log` files in the local directory.  
If the files contain the LDSC result structure, the script identifies only the final result table at the end of each file and concatenates these tables into a single large table.

### Plot.R

Reads the large correlation table generated by `new_table.bash`, transforms it into a correlation matrix, and prepares it for plotting.
